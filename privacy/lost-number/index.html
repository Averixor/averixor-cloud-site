Політика конфіденційності Lost Number

1. Загальна інформація

Lost Number — це мобільна логічна гра. Додаток призначений для гри на пристрої користувача і не потребує створення облікового запису.

Додаток Lost Number не збирає, не зберігає і не передає персональні дані користувачів.

Ми не збираємо:

  • ім’я користувача;
  • адресу електронної пошти;

  • контакти;
  • фото, відео або аудіо;
  • платіжні дані;
  • повідомлення;
  • ідентифікатори пристрою для відстеження користувача.

3. Дані пристрою

Додаток не збирає конфіденційні відомості про пристрій користувача і не використовує їх для ідентифікації або відстеження.

Lost Number не передає персональні дані користувачів третім сторонам, оскільки додаток не збирає такі дані.
